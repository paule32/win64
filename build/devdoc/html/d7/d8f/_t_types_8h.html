◆ FPC_DLLFUNC

#define FPC_DLLFUNC (   x,
  y 
)    _$dll$laz_ ## y ## $ ## x

FPC produce prefix code for each DLL class export; This macro hold proper code for maintain / readable.

It combine the compiler prefix with the class object name.

Parameters
x- function name
y= library name

Possible values for y can be:

  • crt
  • rtl
  • vcl, and many more

Example:

FPC_DLLFUNC(TForm_Create,vcl)
